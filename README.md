<h1>Converter</h1>

<h2>Description</h2>
This project is a simple measurement converter. It can convert between inches and centimetres, feet and metres, and miles and kilometres.
<br />


<h2>Languages and Utilities Used</h2>

- <b>HTML</b> 
- <b>CSS</b>
- <b>JavaScript</b>
- <b>freeCodeCamp</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
